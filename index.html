Download Accelerator

March 2016

A tool built in Java that increases your download speed using dynamic file segmentation. It ensures that your connection is fully used by using Transmission Control Protocol and multi threading.

High Availability Cluster

January 2018

A simple application layer protocol over UDP in Java to facilitate High Availability Cluster (HAC). Implemented a set of mechanism to detect failovers, network/node failures, in order to re-route the traffic to the available systems in both Peer to Peer and Client-Server network.

Web Proxy Server

March 2018

A multithreaded web proxy that supports a subset of functionalities as per RFC 2616 in Java and delivers content on behalf of remote web servers. It also performs caching along with being a forwarding agent. The server however periodically checks for the newer version with the web server and if detects the change then retrieves the latest copy.
